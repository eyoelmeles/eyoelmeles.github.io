`+` one or more

`?` optional so 0 or 1

`*` combine both of them. so 0 or many

`.` it matches everything except for a new line.

`w` word, `W` not a word

`s` whitespace, `S` not a whitespace

`{2, 5}` min and max value

`[a-zA-Z]`

`()` to group and compare
    using
    *   `|`

`^` the very beginning of the line.
